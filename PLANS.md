## 6. Risks

- 周报现在会放宽覆盖数量与抓取上限；如果 prompt 或去重阈值没配平，可能出现“条数增加但重复也增加”的问题。
- 周报增加专题总结后，导出页数和单专题体量会明显大于日报，需要继续观察 PPT / Word 的阅读节奏。
- `overall_insight` 目前仍来自同一轮模型归纳；如果原始热点覆盖不足，周报主题总结也会被源材料上限牵制。
- 当前实现优先做“日报/周报配置分离”，没有单独再造一条完全独立的周报 agent；若后续需求继续提高，可能还需要前移到 query 设计或周报专用 prompt。
- Git 同步仓库与本地运行副本可能存在内容漂移，验证前需要确认关键文件一致。
- 外部搜索接口和模型服务存在时延、额度和偶发失败，验证结果可能带噪声。
- 当前仓库已有未提交改动，若直接大范围格式化或重构，容易引入冲突。
